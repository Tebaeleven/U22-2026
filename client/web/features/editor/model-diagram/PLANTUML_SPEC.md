# PlantUML クラス図 仕様リファレンス

本ドキュメントは、モデル図の描画スタイルを PlantUML に準拠させるための仕様書。

---

## 1. カラーパレット

### デフォルトテーマ（rose.skin）

| 要素 | カラーコード | 説明 |
|------|-----------|------|
| クラス背景 | `#FEFECE` | 薄いクリーム/ペールイエロー |
| 線・境界・矢印 | `#A80036` | ダークレッド/マルーン（**全エッジ共通**） |
| ノート背景 | `#FBFB77` | 明るい黄色 |
| ノート境界 | `#A80036` | 線色と同じダークレッド |
| ページ背景 | `white` | |
| テキスト | `black` | |
| 影 | なし | `Shadowing: 0` |
| 角丸 | なし | `RoundCorner: 0` |

### ステレオタイプ スポットカラー

| エンティティ型 | 文字 | 背景色 |
|------------|------|--------|
| class | **C** | `#ADD1B2`（パステルグリーン） |
| abstract | **A** | `#A9DCDF`（パステルシアン） |
| interface | **I** | `#B4A7E5`（パステルパープル） |
| enum | **E** | デフォルト |

## 2. クラスボックス構造

```
┌───────────────────────┐  ← border: #A80036, 角丸なし
│ <<stereotype>>        │  ← 小文字, テキスト色
│ (C) ClassName         │  ← スポット円(#ADD1B2) + 太字クラス名
├───────────────────────┤  ← 区切り線: #A80036
│ ● +publicField        │  ← 可視性アイコン + フィールド
│ ■ -privateField       │
├───────────────────────┤  ← 区切り線: #A80036
│ ● +publicMethod()     │  ← 可視性アイコン + メソッド
│ ◆ #protectedMethod()  │
└───────────────────────┘
```

- 背景: `#FEFECE`（単色、グラデーションなし）
- 境界: `#A80036`（ダークレッド）
- フォント: SansSerif 14pt
- メンバーフォント: やや小さめ
- abstract クラスの名前はイタリック体

## 3. 可視性アイコン

PlantUML はデフォルトで色付きアイコンを使用する。

| 記号 | 可視性 | アイコン形状 | アイコン色 |
|-----|-------|----------|---------|
| `+` | public | 塗り丸 (●) | 緑 `#00AA00` 系 |
| `-` | private | 塗り四角 (■) | 赤 `#CC0000` 系 |
| `#` | protected | 塗りひし形 (◆) | オレンジ/黄 `#CCAA00` 系 |
| `~` | package | 塗り三角 (▲) | 青 `#0000CC` 系 |

`skinparam classAttributeIconSize 0` でテキスト記号（+, -, #, ~）表示に切替可能。

## 4. 矢印・エッジ仕様

### 関係タイプ一覧

| 関係 | PlantUML記法 | 線スタイル | 始点マーカー | 終点マーカー |
|------|-------------|----------|-----------|-----------|
| 継承/汎化 | `A --|> B` | 実線 | なし | 白抜き三角形 |
| 実現 | `A ..\|> B` | 破線(7,7) | なし | 白抜き三角形 |
| コンポジション | `A *-- B` | 実線 | 塗りひし形 | なし |
| 集約 | `A o-- B` | 実線 | 白抜きひし形 | なし |
| 依存 | `A ..> B` | 破線(7,7) | なし | 開き矢じり(2本線) |
| 関連 | `A --> B` | 実線 | なし | 開き矢じり(2本線) |

### 線色

**全エッジ共通: `#A80036`（ダークレッド/マルーン）**

PlantUML は関係タイプごとに色を変えない。全ての線・矢印が同一色で描画される。

### 矢印形状の SVG 表現

#### 白抜き三角形（継承/実現）
```xml
<polygon fill="#FFFFFF" stroke="#A80036" stroke-width="1"
         points="x1,y1 x2,y2 x3,y3" />
```
- 正三角形に近い二等辺三角形
- 頂点が接続先を指す
- 内部は白塗り

#### 塗りひし形（コンポジション）
```xml
<polygon fill="#A80036" stroke="#A80036" stroke-width="1"
         points="x1,y1 x2,y2 x3,y3 x4,y4" />
```
- 菱形（ダイヤモンド形）
- 塗りつぶし

#### 白抜きひし形（集約）
```xml
<polygon fill="#FFFFFF" stroke="#A80036" stroke-width="1"
         points="x1,y1 x2,y2 x3,y3 x4,y4" />
```
- 菱形
- 内部は白塗り

#### 開き矢じり（依存/関連）
```xml
<path d="M x1,y1 L x2,y2 L x3,y3"
      fill="none" stroke="#A80036" stroke-width="1" />
```
- 2本の線で構成（V字型）
- 閉じない、塗りなし

### 線のルーティング

PlantUML のデフォルトは Graphviz dot のスプライン。ただし、典型的なクラス図の参考画像では**直線セグメント**が使われている。

| linetype 設定 | 描画方法 | 特徴 |
|--------------|---------|------|
| デフォルト（未指定） | スプライン曲線 | 滑らかだが曲がる |
| `ortho` | 直交線（水平・垂直のみ） | 回路図風、重なりやすい |
| `polyline` | 折れ線（直線セグメント） | 直線的、シンプル |

**本プロジェクトでは `straight`（直線）を採用する。** クラス間を最短直線で結び、PlantUML の参考画像に最も近い見た目を実現する。

### ラベル

- 線の中間点付近に配置
- 白背景の矩形で可読性確保
- フォント: SansSerif 11pt
- テキスト色: `#A80036`（線と同色）

### 破線パターン

- PlantUML デフォルト: `stroke-dasharray: 7 7`

## 5. ノート

### 外観

```
┌──────────────────┬┐  ← border: #A80036
│                  ╱│  ← 右上折れ角
│  テキスト内容     │  ← bg: #FBFB77
│                  │
└──────────────────┘
```

- 背景: `#FBFB77`（明るい黄色）
- 境界: `#A80036`（ダークレッド）
- 右上に折れ角（三角形の折り返し効果）
- テキスト: black

### 接続線

- クラスとノートの間は**点線**（`stroke-dasharray: 4 3`）
- 矢印ヘッドなし
- 線色: `#A80036`

### 配置

`note left/right/top/bottom of ClassName` で位置指定。
本プロジェクトでは自由配置（ドラッグ可能）。

## 6. レイアウトアルゴリズム

PlantUML は Graphviz dot の **Sugiyama アルゴリズム**を使用。

### 4フェーズ

1. **サイクル除去**: DFS でバックエッジを検出、一時反転 → DAG化
2. **ランク割り当て**: Network Simplex 法。親クラスが上、子クラスが下
3. **交差最小化**: 重心法（barycenter）+ 中央値法。隣接レイヤー間で交互スイープ
4. **座標割り当て**: Brandes-Köpf アルゴリズム。辺の長さ最小化 + ノード間隔確保

### パラメータ

| パラメータ | 説明 | PlantUML デフォルト |
|----------|------|-------------------|
| rankdir | レイアウト方向 | TB（上→下） |
| nodesep | 同一ランク内のノード間隔 | 約 0.25 inch |
| ranksep | ランク間の距離 | 約 0.5 inch |

### ダッシュ数によるランク距離ヒント

- `--` (2つ): 標準（1ランク差、垂直配置）
- `-` (1つ): 短い（同一ランク寄り、水平配置）
- `---` (3つ以上): 長い（2ランク以上の距離）

## 7. パッケージ/グルーピング

```plantuml
package "パッケージ名" #DDDDDD {
  class ClassA
  class ClassB
}
```

- 背景色指定可能（`#カラーコード`）
- `together { }` で視覚的な囲み線なしの近接配置ヒント

## 8. 区切り線バリエーション

クラス内のセクション間で使用可能な区切り線:

| 構文 | スタイル |
|------|---------|
| `--` | 実線 |
| `..` | 点線 |
| `==` | 二重線 |
| `__` | 太線 |

タイトル付き: `-- Getters --` のように記述可能。
