## 1. イベント

ゲームの開始やキー入力など、「いつ実行するか」を決めるブロック。

### 🟡 旗が押されたとき
**疑似コード:** `onCreate()` メソッド
ゲーム開始時（▶ ボタン押下時）に実行される。

### 🟡 〇キーが押されたとき
**疑似コード:** `onKeyPress("space")`
指定したキーが押されるたびに実行される。
- 選択肢: `space`, `up arrow`, `down arrow`, `right arrow`, `left arrow`, `any`

### 🟡 〇に触れたとき
**疑似コード:** `onTouched("コイン")`
指定したスプライトと衝突したときに実行される。
- 選択肢: `any`（全スプライト）、またはスプライト名

### 🟡 変数〇が変わったとき
**疑似コード:** `onVarChange("score")`
指定した変数の値が変わるたびに実行される。

### 🔵 新しい値 / 古い値
**疑似コード:** `this.newValue` / `this.oldValue`
「変数が変わったとき」ブロック内で使用。変更前後の値を取得する。

### 🟦 変数の監視をやめる
**疑似コード:** `this.unwatch("score")`
変数の監視を停止し、以降の変更イベントを発火しないようにする。

### 🟦 イベントを送る
**疑似コード:** `this.emit("message1", "data")`
名前とデータ付きのカスタムイベントを全スプライトに送信する。

### 🟡 イベント〇を受け取ったとき
**疑似コード:** `onEvent("message1")`
指定したカスタムイベントを受信したときに実行される。

### 🔵 イベントデータ
**疑似コード:** `this.eventData`
受信したイベントに付随するデータを取得する。

## 2. 動き

スプライトの位置・向き・移動を制御するブロック。

### 🟦 〇歩動かす
**疑似コード:** `this.move(10)`
スプライトの向いている方向に指定歩数だけ進む。

### 🟦 〇度回す ↻
**疑似コード:** `this.turnRight(15)`
時計回りに指定角度だけ回転する。

### 🟦 x:〇 y:〇 へ行く
**疑似コード:** `this.setPosition(100, 200)` または `this.goto(100, 200)`
指定した座標へ瞬間移動する。

| 引数 | 説明 |
|------|------|
| x | 横位置。0=中央、正=右、負=左 |
| y | 縦位置。0=中央、正=上、負=下 |

### 🟦 〇秒で x:〇 y:〇 へ滑る
**疑似コード:** `this.glide(1, 100, 200)`
指定時間をかけてスムーズに移動する。実行中はスクリプトが一時停止する。

### 🟦 x座標を〇ずつ変える / y座標を〇ずつ変える
**疑似コード:** `this.changeXBy(10)` / `this.changeYBy(10)`
現在の座標に値を加算する。

### 🟦 x座標を〇にする / y座標を〇にする
**疑似コード:** `this.setX(100)` / `this.setY(200)`（または `this.x = 100`）
座標を直接指定する。

### 🟦 端に着いたら跳ね返る
**疑似コード:** `this.ifOnEdgeBounce()`
ステージの端に触れたとき向きを反転する。

### 🔵 x座標 / 🔵 y座標 / 🔵 向き
**疑似コード:** `this.x` / `this.y` / `this.direction`
現在のx座標・y座標・向きを返す。

### 🟦 Tween で x:〇 y:〇 へ〇秒で移動
**疑似コード:** `this.tweenTo(100, 200, 1)`
イージング付きのスムーズな移動アニメーション。glide と似ているが Phaser の Tween エンジンを使う。

### 🟦 角度を〇にする
**疑似コード:** `this.setAngle(45)` （または `this.angle = 45`）
スプライトの描画角度を設定する。`向き` とは独立した回転。

### 🔵 角度
**疑似コード:** `this.angle`
現在の描画角度を返す。

### 🟦 表示する / 🟦 隠す

スプライトの表示/非表示を切り替える。

### 🟦 コスチュームを〇にする
**疑似コード:** `this.setCostume("costume2")` （または `this.costume = "costume2"`）

### 🟦 次のコスチュームにする

### 🟦 テキストを表示 / 🟦 テキストを更新 / 🟦 テキストを削除
**疑似コード:** `this.addText("Score: 0", -900, 490)` / `this.setText("Score: 10")` / `this.removeText()`
スプライトに紐づくシンプルなHUDテキストの表示・更新・削除。

### 🟦 ID付きテキストを追加
**疑似コード:** `this.addTextAt("score", "SCORE: 0", -900, 490, 32, "#ffffff")`
ID・位置・サイズ・色を指定して高度なテキストを追加する。

| 引数 | 説明 |
|------|------|

| color | 色（`"#ffffff"` 形式） |

### 🟦 ID付きテキストを更新 / 🟦 ID付きテキストを削除
**疑似コード:** `this.updateTextAt("score", "SCORE: 10")` / `this.removeTextAt("score")`

### 🟦 色を〇にする
**疑似コード:** `this.setTint("#ff0000")`
スプライト全体に色フィルタをかける。

### 🟦 色を消す
**疑似コード:** `this.clearTint()`
色フィルタを解除する。

### 🟦 透明度を〇%にする
**疑似コード:** `this.setAlpha(50)`
透明度を 0（透明）〜100（不透明）で設定する。

### 🟦 左右反転を〇にする
**疑似コード:** `this.setFlipX("on")`
スプライトを水平方向に反転する。

### 🟦 浮遊テキスト
**疑似コード:** `this.floatingText("+10")`
スプライトの位置にテキストを表示し、上にフェードアウトする演出。

### 🟦 矩形を描画 / 🟦 描画をクリア
**疑似コード:** `this.graphics.fillRect(0, 0, 100, 20, "#00ff00")` / `this.graphics.clear()`
スプライトに紐づくグラフィックスレイヤーに矩形を描画する。HPバー等に使う。

### 🟦 Tween: スケール / 🟦 Tween: 透明度 / 🟦 Tween: 角度
**疑似コード:** `this.tweenScale(2, 1)` / `this.tweenAlpha(0.5, 1)` / `this.tweenAngle(360, 1)`
指定秒数かけてプロパティをアニメーションする。イージング（Sine.easeInOut）付き。

### 🟦 アニメーション定義
**疑似コード:** `this.createAnim("walk", 0, 3, 10, "on")`
コスチュームの自動切り替えアニメーションを定義する。

| 引数 | 説明 |
|------|------|
| name | アニメーション名 |
| start | 開始コスチュームインデックス（0始まり） |
| end | 終了コスチュームインデックス |
| rate | フレームレート（1秒あたりの切り替え回数） |
| loop | ループ再生するか（"on" / "off"） |

### 🟦 アニメーション再生 / 🟦 アニメーション停止
**疑似コード:** `this.playAnim("walk")` / `this.stopAnim()`

### 🟦 アニメーション完了時にイベント送信
**疑似コード:** `this.onAnimComplete("attack_done")`
ループでないアニメーションの再生完了時にイベントを発火する。

### 🟦 パーティクル発射
**疑似コード:** `this.emitParticles(0, 0, 20, "#ff6600", 200)`
指定座標にパーティクルを爆発的に放出する。

| 引数 | 説明 |
|------|------|
| x, y | 発射位置（ステージ座標） |
| count | パーティクル数 |
| color | 色（`"#ff6600"` 形式） |
| speed | 拡散速度（px/s） |

## 4. 制御

プログラムの流れを制御するブロック。

### 🟦 〇秒待つ
**疑似コード:** `this.wait(1)`
指定秒数だけスクリプトを一時停止する。

### 🔶 〇回繰り返す
**疑似コード:** `repeat(10) { ... }`
中のブロックを指定回数繰り返す。

### 🔶 for (i = 〇 to 〇)
**疑似コード:** `for (i in 1..10) { ... }`
カウンタ変数付きのループ。中で `i` の値を参照できる。

### 🔶 ずっと
**疑似コード:** `while (true) { ... }` または `onUpdate()` メソッド
中のブロックを永遠に繰り返す。

### 🔶 もし〇なら
**疑似コード:** `if (condition) { ... }`
条件が true のとき中のブロックを実行する。

### 🔶 もし〇なら / でなければ
**疑似コード:** `if (condition) { ... } else { ... }`
条件によって実行するブロックを分岐する。

### 🔶 〇まで繰り返す
**疑似コード:** `while (!condition) { ... }`（内部で反転）
条件が true になるまで繰り返す。

### 🟦 〇まで待つ
**疑似コード:** `this.waitUntil(condition)`
条件が true になるまでスクリプトを一時停止する。

### 🟦 すべて止める
**疑似コード:** `this.stop()`
全スクリプトの実行を停止する。

### 🟦 ゲームをリスタート
**疑似コード:** `this.restart()`
ゲームを最初から再開する。

### 🟦 〇のクローンを作る
**疑似コード:** `this.createClone("myself")` または `this.createClone("弾")`
指定スプライトのコピーを作成する。`"myself"` で自分自身のコピー。

### 🟡 クローンされたとき
**疑似コード:** `onClone()`
クローンが作成された直後に実行される。

### 🟦 このクローンを削除する
**疑似コード:** `this.deleteClone()`
このクローンを削除する（元のスプライトには使えない）。

### 🟦 インターバルを設定
**疑似コード:** `this.setInterval("tick", 1000)`
指定ミリ秒ごとにイベントを繰り返し発火する。

### 🟦 インターバルを解除
**疑似コード:** `this.clearInterval("tick")`

### 🟦 タイムアウトを設定
**疑似コード:** `this.setTimeout("delayed", 1000)`
指定ミリ秒後に1回だけイベントを発火する。

## 5. 調べる

入力の検出やスプライト間の関係を調べるブロック。

### 🔷 〇に触れた
**疑似コード:** `this.touching("コイン")`
指定したスプライト・端・マウスポインターに触れているか判定する。
- 選択肢: `mouse-pointer`, `edge`, スプライト名

### 🔷 〇キーが押された
**疑似コード:** `this.isKeyPressed("space")`
指定したキーが**現在押されている**か判定する。押しっぱなしで毎フレーム true。

### 🔷 〇キーが今押された
**疑似コード:** `this.isKeyJustDown("space")`
指定したキーが**今のフレームで押された瞬間**だけ true。連射防止に使う。

| `isKeyPressed` との違い |
|---|
| `isKeyPressed` = 押している間ずっと true（移動に向いている） |
| `isKeyJustDown` = 押した瞬間だけ true（弾の発射に向いている） |

### 🔵 マウスのx座標 / 🔵 マウスのy座標
**疑似コード:** `this.mouseX` / `this.mouseY`
マウスカーソルのステージ座標を返す。

### 🔷 マウスが押された
**疑似コード:** `this.mouseDown`
マウスボタンが押されている間 true。

### 🔵 マウスホイール
**疑似コード:** `this.mouseWheel`
マウスホイールのスクロール量を返す。下=正、上=負。スクロールしていなければ 0。

### 🔵 タイマー
**疑似コード:** `this.timer`
ゲーム開始からの経過秒数を返す。

### 🟦 タイマーをリセット
**疑似コード:** `this.resetTimer()`
タイマーを 0 に戻す。

### 🟦 ドラッグを有効化
**疑似コード:** `this.enableDrag()`
スプライトをマウスでドラッグ移動できるようにする。

## 10. 物理

Phaser Arcade Physics エンジンを制御するブロック。

### 基本設定

#### 🟦 物理モードを〇にする
**疑似コード:** `this.setPhysics("dynamic")`
スプライトの物理エンジンモードを設定する。

| モード | 説明 |
|--------|------|
| `dynamic` | 重力・速度・衝突応答あり。プレイヤーや敵に使う |
| `static` | 位置固定・衝突あり。地面や壁に使う |
| `none` | 物理なし。UI要素や背景に使う |

#### 🟦 重力を〇にする
**疑似コード:** `this.setGravity(500)`
ステージ全体の下方向への重力を設定する（px/s²）。

#### 🟦 個別の重力を〇にする
**疑似コード:** `this.setAllowGravity("off")`
このスプライトだけ重力の影響を受けるかどうかを設定する。
宇宙に浮くスプライトや、UI要素に使う。

---

### 速度（Velocity）

> スプライトの移動速度。正の値=右/下、負の値=左/上。

#### 🟦 速度を x:〇 y:〇 にする
**疑似コード:** `this.setVelocity(200, -300)`
X・Y両方の速度を同時に設定する。

#### 🟦 x速度を〇にする / 🟦 y速度を〇にする
**疑似コード:** `this.setVelocityX(200)` / `this.setVelocityY(-300)`
片方の軸だけ設定する。

#### 🔵 x速度 / 🔵 y速度
**疑似コード:** `this.velocityX` / `this.velocityY`
現在の速度を返す。

#### 🔵 速度の大きさ
**疑似コード:** `this.physicsSpeed`
方向に関係なく、速度の絶対的な大きさ（px/s）を返す。
X=300, Y=400 のとき → √(300²+400²) = 500。

---

### 加速度（Acceleration）

> 速度を毎秒どれだけ変化させるか。車のアクセルに相当。
> velocity が「今のスピード」なら、acceleration は「スピードの増え方」。

#### 🟦 加速度を x:〇 y:〇 にする
**疑似コード:** `this.setAcceleration(200, 0)`

| 値 | 意味 |
|-----|------|
| 正 | 右方向（X）/ 下方向（Y）に加速 |
| 負 | 左方向（X）/ 上方向（Y）に加速 |
| 0 | 加速なし（現在の速度を維持） |

#### 🟦 x加速度を〇にする / 🟦 y加速度を〇にする
**疑似コード:** `this.setAccelerationX(-600)` / `this.setAccelerationY(100)`

**velocity と acceleration の違い:**
```
setVelocityX(200)       → 即座に 200px/s で右へ移動
setAccelerationX(200)   → 0 → 200 → 400 → 600... と加速していく
```

---

### 抗力とダンピング（Drag / Damping）

> 入力がないとき自然に減速する力。摩擦や空気抵抗。

#### 🟦 抗力を x:〇 y:〇 にする
**疑似コード:** `this.setDrag(300, 300)`

#### 🟦 ダンピングを〇にする
**疑似コード:** `this.setDamping("on")`

| モード | drag の意味 | 動き方 |
|--------|-----------|--------|
| off（通常） | px/s²（毎秒減る量） | drag=300 → 毎秒300px/sずつ減速 |
| on（ダンピング） | 0〜1の割合（毎フレーム掛ける割合） | drag=0.99 → 毎フレーム速度が99%に |

**使い分け:**
- 地上キャラ → 通常モード（`setDrag(300, 300)`）
- 宇宙船 → ダンピング（`setDamping("on")` + `setDrag(0.99, 0.99)`）

---

### 最大速度

#### 🟦 最大速度を x:〇 y:〇 にする
**疑似コード:** `this.setMaxVelocity(300, 300)`
速度の上限。加速度で加速し続けても超えない。デフォルトは 10000（実質無制限）。

---

### 角速度（Angular Velocity）

> 回転する速さ（度/秒）。

#### 🟦 角速度を〇にする
**疑似コード:** `this.setAngularVelocity(180)`

| 値 | 意味 |
|-----|------|
| 正 | 時計回り |
| 負 | 反時計回り |
| 0 | 回転停止 |

**setAngle との違い:**
- `setAngle(45)` → 即座に45度にジャンプ
- `setAngularVelocity(180)` → 毎秒180度ずつなめらかに回転

---

### 衝突プロパティ

#### 🟦 反発係数を〇にする
**疑似コード:** `this.setBounce(0.8)`
衝突時の跳ね返り。0=跳ねない、1=完全に跳ね返る。

#### 🟦 ワールド境界衝突を〇にする
**疑似コード:** `this.setCollideWorldBounds("on")`
ステージ端で止まる/跳ね返るようにする。

#### 🟦 不動体を〇にする
**疑似コード:** `this.setImmovable("on")`
衝突で押されなくなる。ただし自分は velocity で移動できる。
動く床やパドルに使う。

| | static モード | immovable |
|---|---|---|
| 押されない | ✅ | ✅ |
| 自分で動ける | ❌ | ✅ |
| 重力の影響 | ❌ | 設定次第 |

#### 🟦 質量を〇にする
**疑似コード:** `this.setMass(5)`
衝突の反発に影響。重いほど押されにくい。デフォルト 1。

#### 🟦 押し出し可否を〇にする
**疑似コード:** `this.setPushable("off")`
off にすると他スプライトに押されなくなる（immovable と似ているが、より細かい制御）。

#### 🔷 地面に着いている？
**疑似コード:** `this.isOnGround()`
下方向に接触しているか。ジャンプ判定に使う。

#### 🟦 衝突コールバック登録
**疑似コード:** `this.onCollide("敵", "hit-enemy")`
指定スプライトとの衝突時にイベントを発火する。

#### 🔵 衝突相手
**疑似コード:** `this.collisionTarget`
衝突イベント内で、衝突した相手の名前を返す。

#### 🟦 ボディを無効化 / 🟦 ボディを有効化
**疑似コード:** `this.disableBody()` / `this.enableBody()`
物理ボディを無効化すると非表示＋衝突無効になる。

### 高度な移動

#### 🟦 〇に向かって速度〇で移動
**疑似コード:** `this.moveTo("プレイヤー", 200)`
指定スプライトに向かって一定速度で移動する。追跡AIに使う。

#### 🟦 〇に向かって加速度〇で加速
**疑似コード:** `this.accelerateTo("プレイヤー", 100)`
指定スプライトに向かってじわじわ加速する。moveTo より自然な追跡。

#### 🟦 角度〇から速度〇を設定
**疑似コード:** `this.velocityFromAngle(this.angle, 200)`
角度と速度から velocity の x/y 成分を計算して設定する。
「向いている方向に進む」動きに使う。

```
角度 0°  → vx=200, vy=0  （右）
角度 90° → vx=0, vy=200  （下）
角度 -90°→ vx=0, vy=-200 （上）
```

#### 🟦 ワールドラップ
**疑似コード:** `this.worldWrap(32)`
画面端を超えたら反対側から出現する。パックマン風の動き。
`onUpdate()` 内で毎フレーム呼ぶ必要がある。

## 座標系まとめ

```
              y = +540（上端）
                  |
 x = -960 ────── 0,0 ────── x = +960
（左端）          |           （右端）
              y = -540（下端）
```

- ステージサイズ: 1920 × 1080 px
- 中心: (0, 0)
- X: 右が正、左が負
- Y: **上が正、下が負**（数学座標系。Phaser とは逆）

## 角度の座標系

```
     0° = 右
    90° = 下
   180° = 左
   -90° = 上
```

`velocityFromAngle` や `angleTo` はこの座標系を使う。
`setAngle` で設定する描画角度も同じ。
