# エディタ画面改善提案

Scratch / TurboWarp / Unity / Unreal Engine / Godot / MakeCode / Construct 3 / GDevelop / RPG Maker の
UI/機能を調査し、現状に足りない機能を優先順位ごとに整理したドキュメント。

---

## 現状のエディタ構成


| パネル       | 内容                            |
| --------- | ----------------------------- |
| palette   | ブロックパレット（カテゴリ別）               |
| workspace | コード / コスチューム / コライダー（3タブ）     |
| stage     | Phaser ステージ（プレビュー + 実行）       |
| sprites   | スプライトリスト                      |
| hierarchy | シーンツリー                        |
| inspector | プロパティエディタ                     |
| debug     | AST / スプライト / 変数 / VM / 疑似コード |
| samples   | サンプルブラウザ                      |


**既にある機能**: 再生/一時停止/停止、スプライト管理、コスチューム描画、コライダー編集、デバッグパネル、サンプル読み込み、プロジェクト保存、FPS表示

---

## P0: 基本的な操作性（すぐ必要）

### 0-1. Undo / Redo

- **参考**: Scratch (Ctrl+Z / Ctrl+Shift+Z)、全エディタ共通
- **現状**: headless-vpl に History モジュールはあるが UI に未接続
- **対応**: ツールバーにボタン追加 + キーボードショートカット接続
- **コスト**: 低 / **インパクト**: 極大

### 0-2. ブロック右クリックメニューの充実

- **参考**: Scratch（複製、削除、コメント追加、ヘルプ）
- **現状**: 右クリックメニューは存在するが最小限
- **対応**: 「複製」「全削除（この下のブロック全て）」「無効化（コメントアウト）」を追加
- **コスト**: 低 / **インパクト**: 大

### 0-3. ブロック検索（Ctrl+Space / 中クリック）

- **参考**: TurboWarp / Scratch Addons の「Insert blocks by name」
- **現状**: なし。ブロック数が増えると目的のブロックを見つけにくい
- **対応**: ワークスペース上でポップオーバー検索 → ブロック挿入
- **コスト**: 中 / **インパクト**: 大

### 0-4. キーボードショートカット一覧 / ヘルプ

- **参考**: 全エディタ共通（? キーでショートカット一覧表示）
- **現状**: ショートカットは存在するが一覧がない
- **対応**: ヘルプダイアログ or ツールチップ
- **コスト**: 低 / **インパクト**: 中

---

## P1: ゲーム開発に必須の機能

### 1-1. サウンドエディタ / サウンドタブ

- **参考**: Scratch（スプライトごとに音声タブ）、Unity（AudioSource コンポーネント）
- **現状**: workspace に「コード / コスチューム / コライダー」の3タブのみ
- **対応**: 4つ目のタブ「サウンド」追加。効果音・BGM の追加/再生/削除
- **コスト**: 中 / **インパクト**: 大

### 1-2. ステージの全画面プレイモード

- **参考**: Scratch（全画面ボタン）、TurboWarp（カスタムステージサイズ）
- **現状**: ステージ拡大ダイアログはあるが、プレイ専用の全画面モードがない
- **対応**: ステージのみを画面いっぱいに表示するフルスクリーンモード
- **コスト**: 低 / **インパクト**: 大

### 1-3. スプライト複製

- **参考**: Scratch（右クリック→複製）、Unity（Ctrl+D）
- **現状**: スプライトの追加/削除のみ、複製がない
- **対応**: スプライトリストとヒエラルキーで「複製」ボタン。ブロックデータも含めてコピー
- **コスト**: 低 / **インパクト**: 大

### 1-4. 実行速度制御（ターボモード / カスタム FPS）

- **参考**: TurboWarp（カスタム FPS）、Scratch（Shift+旗クリックでターボモード）
- **現状**: 60FPS 固定
- **対応**: ステージ横に速度スライダー or ヘッダーに FPS 設定
- **コスト**: 低 / **インパクト**: 中

### 1-5. ステップ実行 / ブレークポイント

- **参考**: Unreal Blueprint（ブレークポイント + ステップ実行）、Unity（Pause + Step）
- **現状**: デバッグパネルで実行中ブロックのハイライトはある
- **対応**: ブロックにブレークポイント設置 → 到達時に自動一時停止 → ステップ実行ボタン
- **コスト**: 高 / **インパクト**: 中

## P2: 開発効率の向上

### 2-1. 変数ウォッチ / ステージ上の変数モニター

- **参考**: Scratch（ステージ上に変数値をリアルタイム表示）、Unreal（Watch ウィンドウ）
- **現状**: デバッグパネルの「変数」タブでのみ確認可能
- **対応**: ステージ上にオーバーレイとして変数値を表示するオプション
- **コスト**: 中 / **インパクト**: 中

### 2-2. コメントボックス

- **参考**: Scratch（ブロックに吹き出しコメント）、Unreal Blueprint（コメントボックスでノード群をグループ化）
- **現状**: なし
- **対応**: ワークスペース上にテキストボックスを配置できる機能
- **コスト**: 中 / **インパクト**: 中

### 2-3. ブロックの「クリーンアップ」

- **参考**: Scratch（右クリック → "Clean up Blocks" でスクリプトを整列）
- **現状**: なし
- **対応**: ワークスペースの右クリックメニューに追加。スクリプトを縦に整列
- **コスト**: 低 / **インパクト**: 中

### 2-4. バックパック（ブロックの保存と再利用）

- **参考**: Scratch Backpack（スクリプト / コスチューム / 音を保存して他プロジェクトで再利用）
- **現状**: なし
- **対応**: パレット下部にバックパック領域。ブロック群をドラッグして保存
- **コスト**: 高 / **インパクト**: 中

### 2-5. コンソール / ログパネル

- **参考**: Unity Console、ブラウザ DevTools
- **現状**: デバッグパネルの VM タブが部分的に代替
- **対応**: `print` ブロックの出力先となるコンソールパネル
- **コスト**: 低 / **インパクト**: 中

### 2-6. アセットブラウザ

- **参考**: Unity Project Browser、Godot FileSystem Dock
- **現状**: コスチュームは各スプライトの「コスチューム」タブで管理
- **対応**: プロジェクト全体の画像・音声アセットを一覧管理するパネル
- **コスト**: 中 / **インパクト**: 中

## P3: UX 向上

### 3-1. ツールチップ / ブロックのヘルプ

- **参考**: Scratch（ブロック右クリック → ヘルプ）、MakeCode（ホバーでドキュメント表示）
- **対応**: ブロックホバー時にツールチップで説明表示
- **コスト**: 低 / **インパクト**: 小

### 3-2. スプライトのドラッグ並べ替え

- **参考**: Scratch（スプライトリストでドラッグ順序変更）
- **対応**: ドラッグ&ドロップで順序変更（描画順にも影響）
- **コスト**: 低 / **インパクト**: 小

### 3-3. レイアウトプリセット

- **参考**: Unity（Default / 2x3 / 4 Split 等のプリセット）
- **対応**: 「コーディング向け」「デバッグ向け」「デザイン向け」等のプリセット
- **コスト**: 低 / **インパクト**: 小

### 3-4. ダーク / ライトテーマ切り替え

- **参考**: TurboWarp、Unity、VS Code
- **対応**: ヘッダーにテーマ切り替えトグル
- **コスト**: 中 / **インパクト**: 小

### 3-5. ミニマップ

- **参考**: VS Code ミニマップ、Unreal Blueprint の概要表示
- **対応**: ワークスペース右下にミニマップ表示
- **コスト**: 中 / **インパクト**: 小

### 3-6. オニオンスキン（コスチュームエディタ）

- **参考**: Scratch Addons、アニメーションツール全般
- **対応**: コスチュームタブで前後フレームを半透明表示
- **コスト**: 中 / **インパクト**: 小

## P4: 拡張性

### 4-1. カスタムブロック拡張（エクステンション）

- **参考**: Scratch Extensions、TurboWarp Extension Gallery
- **対応**: JavaScript で書いたカスタムブロックをロードする仕組み
- **コスト**: 高 / **インパクト**: 中

### 4-2. タイルマップエディタ

- **参考**: MakeCode Arcade（タイルベースレベルシステム）、RPG Maker、Construct 3
- **対応**: ステージ上でタイルを配置してレベルデザインできるエディタ
- **コスト**: 高 / **インパクト**: 大

### 4-3. アニメーションタイムライン

- **参考**: Unity Animation Window、Construct 3 Timeline
- **対応**: キーフレームベースのアニメーションエディタ
- **コスト**: 高 / **インパクト**: 中

### 4-4. プロジェクトのエクスポート / パッケージング

- **参考**: TurboWarp Packager（スタンドアロンアプリ化）
- **対応**: HTML5 / スタンドアロンとしてエクスポート
- **コスト**: 高 / **インパクト**: 大
